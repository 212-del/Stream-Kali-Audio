# Stream-Kali-Audio
Lightweight Bash utility that captures Kali Linux system audio and streams it live to VLC over the network. Uses PulseAudio/PipeWire and standard CLI tools to create a low-latency audio stream for monitoring, demos, or remote listening. Simple setup, minimal dependencies, configurable ports, codecs, and streaming options.

# Steps to Install
*git clone https://github.com/212-del/Stream-Kali-Audio.git*

*chmod +x Stream.sh*

*bash Stream.sh*


# One liner
git clone https://github.com/212-del/Stream-Kali-Audio.git && chmod +x Stream.sh && bash Stream.sh
